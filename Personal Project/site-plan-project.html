Color Scheme

Color Name Hex Code Usage

Typography

Heading Font: Georgia

Georgia will be used for all headings (h1–h6) to give a professional and classic look.

Body Font: Arial

Arial will be used for paragraphs and general content because it is clean and easy to read.

Typography Hierarchy

  • h1 – Main page title
  • h2 – Section titles
  • h3 – Subsection titles
  • p – Regular paragraph text
